Belajar Membuat Website

Internet adalah cara paling efektif untuk berbagi dan menyebarluaskan ilmu. Di SUKASUKA.com anda bisa menemukan artikel artikel menarik seputar pemograman dasar juga contoh-contoh pemograman web dengan PHP,HTML,CSS,Javascript dan framework seperti bootstrap

HTML

HTML adalah bahasa pemograman dasar untuk membuat sebuah website. HTML wajib dipelajari jika anda ingin belajar web, baik itu desain web ataupun pemograman web secara menyeluruh.

CSS

CSS adalah salah satu cara untuk mengkostumasi tampilan web agar tak terkesan monoton sehingga menarik bagi pembaca dan pengunjung website kita. sebagai seorang desainer web, CSS wajib dipelajari dan dikuasai.

Bootstrap

Bootstrap adalah sebuah framework siap pakai yang terdiri dari kumpulan styling CSS, fungsi javascript, dan Jquery. Anda dapat mendesain website dengan lebih terorganisir dan mudah menggunakan framework Bootstrap. Bootstrap memberikan kemudahan desain, dengan upaya yang lebih sedikit karena memiliki banyak fungsi umum yang siap pakai untuk desain web anda.
